# Calculadora de IMC - JavaFX

## 📋 Descrição
Aplicação desenvolvida em JavaFX para calcular o Índice de Massa Corporal (IMC) 
com persistência de dados em arquivo.

## 🛠️ Tecnologias
- Java 
- JavaFX
- Maven

## ▶️ Como executar
1. Abrir o projeto em uma IDE (IntelliJ IDEA ou Eclipse)
2. Executar a classe `Launcher.java`
3. Ou via terminal: `mvn javafx:run`

## 📁 Estrutura do Projeto
- `IMCApp.java` - Classe principal JavaFX
- `Pessoa.java` - Modelo de dados
- `ArquivoUtil.java` - Manipulação de arquivos
- `HelloController.java` - Controlador da interface
- `main-view.fxml` - Interface gráfica
